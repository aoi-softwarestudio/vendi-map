# Agent Skill: Performance Optimization

## Purpose
システム全体のパフォーマンスボトルネックを検出し、応答速度とリソース効率を最大化するスキル。

---

## 📋 Execution Checklist
Phase 1: ボトルネックの特定
1. プロファイリングツールの実行
2. メモリリークの監査

Phase 2: 最適化の実行
1. キャッシュレイヤーの導入
2. データベースクエリのチューニング

Phase 3: 効果検証
1. 負荷テストの実施
2. レスポンスタイムの比較測定

---

## 📝 Deliverable Format
最適化実績レポート（計測データ、適用パッチ、およびチューニング前後の比較グラフを含むMarkdownファイル）
